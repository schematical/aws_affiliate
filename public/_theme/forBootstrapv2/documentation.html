Shopfrog documentation

Here you will find answers on how to configure your Shopfrog theme.

Selecting the style

Shopfrog is available in seven different styles: pink, orange, brown, green, blue, grey and black and white.

Once you have chosen the style you want all you need to do is set the correct stylesheet in the head of each file.

Currently the theme is set up as pink and will look like this:

<link href="css/shopfrog-pink.css" rel="stylesheet" media="screen">

<!--<link href="css/shopfrog-blue.css" rel="stylesheet" media="screen">
<link href="css/shopfrog-brown.css" rel="stylesheet" media="screen">
<link href="css/shopfrog-green.css" rel="stylesheet" media="screen">
<link href="css/shopfrog-orange.css" rel="stylesheet" media="screen">
<link href="css/shopfrog-grey.css" rel="stylesheet" media="screen">
<link href="css/shopfrog-bw.css" rel="stylesheet" media="screen">-->

Delete all other lines other than the theme style you have chosen. For example, orange:

<link href="css/shopfrog-orange.css" rel="stylesheet" media="screen">

Using a bigger/smaller logo

If your name/logo is wider than Shopfrog, you will notice it breaks the layout slightly.

To fix this alter the values of the following three css declarations to make the appropriate elements bigger to better accommodate your name.

.navbar .brand {
	width: 235px;
}	

.submenu-container .submenu {
	width: 255px;
}	

.submenu-container .submenu-preview {
	left: 255px;
}

Configuring filters on product boards

Adding filters to a product board page (collection.html) is quite simple. A class on the product is paired with a data attribute on the link.

<a href="" data-filter="cat-1">black</a>
...
<div class="product medium cat-1">...</div>
